# Images — AJ's Diamond Detailing

Drop your real photos in this folder and update the `src` paths in `index.html`
(every placeholder image points to placehold.co right now). Use **descriptive,
hyphenated, lowercase filenames** — they're an SEO signal and the `alt` text
already references Gainesville/detailing.

| Filename (recommended)              | Where it's used            | Recommended size      | What to show |
|-------------------------------------|----------------------------|-----------------------|--------------|
| `hero-detailed-car-gainesville.jpg` | Hero background            | 1920×1080 (landscape) | Your best wide shot of a freshly detailed car — wet gloss, reflections |
| `before-interior.jpg`               | Before/after slider (left) | 1200×720              | A dirty interior **before** you started |
| `after-interior.jpg`                | Before/after slider (right)| 1200×720              | The **same** interior after detailing (must match the before shot) |
| `exterior-gloss.jpg`                | Gallery                    | 600×600 (square)      | Glossy paint / hand wax reflection |
| `wheels.jpg`                        | Gallery                    | 600×600               | Clean wheels + tire shine |
| `interior.jpg`                      | Gallery                    | 600×600               | Steam-cleaned dashboard / interior |
| `leather.jpg`                       | Gallery                    | 600×600               | Conditioned leather seats |
| `engine-bay.jpg`                    | Gallery                    | 600×600               | Detailed engine bay |
| `ceramic-beading.jpg`               | Gallery                    | 600×600               | Water beading on a ceramic-coated panel |
| `meet-aj.jpg`                       | About section              | 700×800 (portrait)    | Owner / team at work |
| `storefront.jpg`                    | Schema `image` (SEO)       | 1200×800              | Van, shop, or branded photo |
| `social-share.jpg`                  | Open Graph share image     | 1200×630              | Branded image shown when the link is shared on social/text |

## Tips
- Compress images before uploading (TinyPNG / Squoosh) — big photos slow the site.
- JPG for photos, PNG only for logos/graphics with transparency.
- Keep the before/after pair shot from the **same angle** for the slider to look right.
